Positioning

Positioning is a very important concept in web development. It allows us to place elements in a specific position on the web page. There are different types of positioning in CSS. They are:

Static
( position: static; top: 0px; background-color: rgb(255, 106, 0); padding: 50px; font-size: 20px;)

Relative
( position: relative; top: -50px; background-color: green; padding: 50px; font-size: 20px;)

Absolute
( position: absolute; top: 100px; right: 90px;
background-color: blue; padding: 20px; font-size: 20px;)
